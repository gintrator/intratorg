Bezier Surface Tessellation

Uniform and adaptive tessellation. Wireframe and filled models. Flat and smooth shading. The model is rendered using OpenGL and can be rotated and translated using the keyboard. Using uniform tessellation, the patches are a constant step size given as input argument. Using adaptive tessellation, the patches are calculated with enough precision so that the total curvature is less than a given input argument.

RayTracer

Implements the ray tracing algorithm. Supports sphere and triangle primitives, direct and point lights, transforms as well as .obj files. Lighting is calculated using an implementation of Phong shading, where the total illumination is equal to the sum of the ambient, diffuse, and specular components. Supports an input scene format allowing the user to specify lights, materials, shapes, transforms, and .obj files and outputs to .png format images.

Inverse Kinematics

Given a goal position in 3d space and an n-length posable arm, calculates the angles required at each joint to reach the goal position. Uses jacobians and singular value decomposition to closely approximate this calculation. Iteratively improves by reducing the angle step if the arm is too far from the goal position. The model is rendered using OpenGL and can be rotated and translated using the keyboard.
